# 🎬 Plot Director

**Plot Director** — расширение для [SillyTavern](https://github.com/SillyTavern/SillyTavern), которое работает как помощник режиссёра: генерирует пошаговый сюжетный план, скармливает его модели по одному шагу, и автоматически переключает шаги по мере продвижения истории.

Решает главную проблему ролевых чатов с LLM — модель не умеет вести долгосрочный сюжет. Если дать ей весь план, она попытается выполнить его за два ответа. Plot Director выдаёт ровно один шаг за раз, а антиспойлер не даёт вам, игроку, прочитать, что будет дальше.

---

## Возможности

- **Генерация сюжета** через API с полным контекстом (чат, лорбук, карточки персонажей, персона)
- **Пошаговая инъекция** текущего шага через механику Author's Note (depth 1, не ломает prompt caching)
- **Автодетекция завершения** — модель ставит маркер `[step N complete]`, расширение его вырезает и переходит к следующему шагу
- **Антиспойлер** — шаги генерируются на незнакомом вам языке (корейский, японский, китайский, арабский и др.)
- **Спойлер-блюр** — если шаги на знакомом языке, текст размыт до клика
- **Откат при свайпе** — если свайпнутый ответ не содержит маркера, шаг откатывается
- **Перегенерация хвоста** — вручную или автоматически при каждом переходе к новому шагу
- **Ручное редактирование** любого шага
- **Навигация** — прыжок к любому шагу, ручное переключение вперёд/назад

## Настройки генерации

| Параметр | Описание |
|----------|----------|
| Количество шагов | 2–20 |
| Временной охват | Сцена, день, неделя, месяц, год, десятилетие |
| Язык | Антиспойлерный язык для описания шагов |
| Жанр | Drama, Thriller, Romance, Mystery, Adventure, Horror, Comedy, Sci-Fi, Fantasy, Slice of Life |
| Эпичность | 1–10 (от камерной до эпической) |
| Реализм | 1–10 (от фантастического до приземлённого) |
| Авторское направление | Свободное поле для указания желаемого вектора сюжета |

## Установка

### Через интерфейс SillyTavern (рекомендуется)

1. Откройте SillyTavern
2. Перейдите в **Extensions** → **Install Extension**
3. Вставьте URL этого репозитория:
   ```
   https://github.com/YOUR_USERNAME/plot-director
   ```
4. Нажмите **Install**
5. Расширение появится в панели Extensions как **🎬 Plot Director**

### Вручную

```bash
cd SillyTavern/data/default-user/extensions/third-party/
git clone https://github.com/YOUR_USERNAME/plot-director.git
```

Перезапустите SillyTavern.

## Использование

### Быстрый старт

1. Откройте чат с персонажем
2. В панели Extensions найдите **🎬 Plot Director**
3. Нажмите **New** — откроется окно настроек
4. Выберите параметры и нажмите **Generate**
5. Расширение сгенерирует план и начнёт инжектить первый шаг
6. Играйте как обычно — модель будет вести сюжет к текущему шагу
7. Когда шаг выполнен, модель поставит маркер, расширение переключится на следующий

### Панель управления

| Кнопка | Действие |
|--------|----------|
| **New** | Сгенерировать новый сюжетный план |
| **Steps** | Открыть список шагов (просмотр, редактирование, навигация) |
| **Regen** | Перегенерировать оставшиеся шаги с учётом текущего контекста |
| **◀** | Откатить на предыдущий шаг |
| **▶** | Перейти к следующему шагу вручную |
| **Clear** | Удалить текущий план |

### Переключатели

- **Injection active** — включает/выключает инъекцию шага в промпт
- **Auto-regenerate tail** — автоматически перегенерирует оставшиеся шаги при каждом переходе

### Маркер завершения

Модель должна поставить маркер `[step N complete]` в конце ответа, когда текущий шаг сюжета выполнен. Расширение автоматически:
- Обнаружит маркер
- Вырежет его из отображаемого текста
- Переключится на следующий шаг

Маркер регистронезависимый: `[step 3 complete]`, `[Step 3 Complete]`, `[STEP 3 COMPLETE]` — все работают.

## Технические детали

- Данные сюжета хранятся в `chat_metadata` — привязаны к конкретному чату
- Инъекция через `setExtensionPrompt()` на depth 1 (эквивалент Author's Note)
- Генерация через `generateQuietPrompt()` — использует текущий API, модель и контекст пользователя
- Не добавляет сообщений в чат при генерации/регенерации плота
- Совместим с prompt caching (инъекция на depth 1 не затрагивает cached zone на depth 2+)

### Требования

- SillyTavern (staging branch рекомендуется)
- Любой LLM-бэкенд (Claude, OpenAI, и т.д.)

### Известные нюансы

- Сигнатуры событий `MESSAGE_RECEIVED` / `MESSAGE_SWIPED` могут различаться между версиями ST — если авто-детекция не срабатывает, проверьте консоль (`F12 → Console → [Plot Director]`)
- При первом запуске может потребоваться перезагрузка страницы для корректной инициализации
- `generateQuietPrompt` с полным контекстом занимает время и токены — это нормально
